Introduction

Matplotlib是python最出名的面向对象绘图库。官网：matplotlib.org, 成品展示：gallery

pylab

pylab 是 matplotlib 面向对象绘图库的一个接口。它的语法和 Matlab 十分相近。也就是说，它主要的绘图命令和 Matlab 对应的命令有相似的参数。
