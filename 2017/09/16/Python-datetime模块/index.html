Python datetime模块

发表于 2017-09-16 | 分类于 Algorithm and Computer Science

datetime模块提供大量处理日期、时间、时间间隔的方法。

所有提供的type都是新风格的类，可以从python中被继承和拓展。模块包括以下type：

  • datetime：一天中的日期和时刻
  • date：只是1-9999年的日期
  • time：代表时刻，独立于日期
  • timedelta：时间间隔，用来操纵日期
  • tzinfo：用于为time／datetime对象实现时区支持

the datetime type

datetime对象代表某时区中的日期和时刻。除非特别指明，默认使用’naive time’，即取决于所在应用的时区。datetime也提供时区支持，以后再写。

对于date和datetime，python提供了获取指定日期是星期几的方法weekday()/isoweekday()。两者区别在于礼拜的索引从0还是从1开始：

date.weekday():

Return the day of the week as an integer, where Monday is 0 and Sunday is 6. For example, date(2002, 12, 4).weekday() == 2, a Wednesday. See also isoweekday().

date.isoweekday():

Return the day of the week as an integer, where Monday is 1 and Sunday is 7. For example, date(2002, 12, 4).isoweekday() == 3, a Wednesday. See also weekday(), isocalendar().
